# Cursor / Windows SSH 密钥连接指南

以后默认分工：

| 谁 | 做什么 |
| --- | --- |
| **你** | 只负责：把公钥（`.pub` 那一行）拷到服务器 `authorized_keys`（或宝塔「SSH 密钥」里贴一下） |
| **我（Agent）** | 生成本机密钥、写 `~/.ssh/config`、需要时帮你核对能否免密 / Cursor 怎么连 |

你不必自己记 `ssh-keygen` 全套命令。新服务器时直接说：「帮我配 SSH，IP 是 xxx」，我做完后只把**要粘贴的那一行公钥**给你。

---

## 你唯一要做的一步（拷到服务器）

我生成好密钥后，会给你类似这样的**一整行**公钥：

```text
ssh-ed25519 AAAAC3...（很长一串） cursor-xxx
```

任选一种贴到服务器即可：

### 方式 A：宝塔 / 面板（最省事）

面板 → SSH 密钥 / 安全 → 粘贴整行 → 保存。

### 方式 B：已能登录服务器时

```bash
mkdir -p ~/.ssh && chmod 700 ~/.ssh
echo '这里粘贴我给你的整行公钥' >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
```

贴完告诉我一声，我帮你验证 `ssh 别名` 是否免密；通过后 Cursor 里选该别名连接即可。

---

## Cursor 怎么连（配好之后）

1. `F1` → **Remote-SSH: Connect to Host...**
2. 选我写好的别名（例如本项目：`jzx-prod`）
3. 打开服务器目录（本项目：`/opt/jizhuangxiang`）

## 安全提醒

- **私钥不要**发聊天、不要上传公开处  
- 聊天里若发过 root 密码，请在服务器执行 `passwd` 更换  
- 改密码**不影响**已配好的密钥登录
