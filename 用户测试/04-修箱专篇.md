## 4. 状态机速查

```
待报修 ──提交报修──→ 待检验 ──检验通过·派修──→ 维修中 ──完工报验(录 actualCost)──→ 待验收 ──验收通过──→ 已完工
                      │                              │
                      └──────── 判废（报废评估） ──────┴→ 已报废
```

| 状态 | 界面按钮 | 副作用 |
| :--- | :--- | :--- |
| 待报修 | **提交报修** | → 待检验 |
| 待检验 | **检验通过·派修** / **判废** | → 维修中；箱主档 **维修中**；available−1 |
| 维修中 | **完工报验** | 录入实际维修费 → 待验收 |
| 待验收 | **验收通过** | → 已完工；箱主档 **在场**；available+1 |
| 待检验/维修中 | **判废** | → 已报废；onSite−1 |

费用：无独立修箱账单资源；`estCost` / `actualCost` 在工单上；页顶可看累计维修费。
